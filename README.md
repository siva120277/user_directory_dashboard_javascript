# User Directory Dashboard

A simple User Directory Dashboard built using HTML, CSS, and JavaScript. This project fetches user and post data from the JSONPlaceholder API and displays it dynamically. It also includes a live search feature to filter users by name.

## 🚀 Features

- Load user details from an API
- Display user name, email, and phone number
- Load and display posts
- Live search to filter users
- Dynamic DOM manipulation
- Responsive and simple user interface

## 🛠️ Technologies Used

- HTML5
- JavaScript (ES6)
- Fetch API
- JSONPlaceholder API
- DOM Manipulation

## 📚 Concepts Practiced

- Fetch API
- Async/Await
- Event Listeners
- DOM Manipulation
- Live Search
- Arrays and Objects
- Dynamic Content Rendering

## 🎯 Learning Outcome

This project helped me understand how to fetch data from APIs using JavaScript, work with asynchronous programming using async/await, dynamically update the DOM, and implement a live search feature for filtering user data.
